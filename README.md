# Project: Vue + Vuetify (JS) Forecast-app

Application displays a 7-day precipitation forecast based on user search.

## Tech Stack
- **Vue 3** (Vite)
- **Vuetify 3** (Material Design)
- **Vue Router** (Routing)
- **Pinia** (State management)
- **Open-Meteo API** (Weather and location data)

## Requirements

- **Node.js** (v22.21.1 or newer)
- **pnpm** (package manager)

## How to run

### 1. Clone repository

```bash
git clone https://github.com/PauliMaenpaa/forecast-app
```

### 2. Move to project folder

```bash
cd forecast-app
```
### 3. Open project

```bash
code .
```

### 4. Install dependencies

```bash
pnpm install
```

### 5. Run as local host

```bash
pnpm run dev
```
